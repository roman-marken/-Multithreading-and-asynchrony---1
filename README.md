## Відповідність ТЗ

1. **Завдання 1** — один `Thread` виводить числа **0…50** у консоль.
2. **Завдання 2** — той самий підхід; користувач задає **початок і кінець** діапазону.
3. **Завдання 3** — користувач задає **кількість потоків** і межі; діапазон ділиться між потоками, рядки підписані номером потоку.
4. **Завдання 4** — масив **10000** цілих; **три потоки** окремо знаходять максимум, мінімум і середнє арифметичне; після `Join` результати виводяться в консоль.
5. **Завдання 5** — як 4, додатково **окремий потік** записує весь набір і результати у **текстовий файл** (UTF-8; шлях вводиться або за замовчуванням у каталозі програми).
